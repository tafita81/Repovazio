Como criar os grupos (8 temas)

1
Criar grupo no WhatsApp → Novo grupo → Adicionar participantes → Nome do grupo (ex: "ψ Apego Ansioso — Daniela Coelho")
2
Configurar grupo → Foto de capa (baixar em /brand.html) → Descrição → Apenas admins enviam mensagens
3
Copiar link de convite → Informações do grupo → Link de convite do grupo → Copiar
4
Salvar link no banco:
UPDATE whatsapp_groups_temas SET invite_link = 'https://chat.whatsapp.com/SEU_LINK' WHERE tema = 'apego_ansioso';
5
Divulgar → Adicionar link na bio do Instagram/YouTube → Compartilhar nos primeiros vídeos
