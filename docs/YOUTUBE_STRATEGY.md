## 🎯 PASSO 1: Escolher o Nicho

### Dois critérios inegociáveis:
1. **Paixão + Expertise** — sem isso, os vídeos ficam ruins e você não mantém consistência
2. **TAM (Total Addressable Market)** — qual o máximo de pessoas que PODEM assistir?

### Prompt exato para Claude:
```
Estou pensando em criar um canal no YouTube sobre [SEU NICHO].
Quais são 5 problemas que posso resolver para esse público,
e qual o tamanho desse público-alvo?
```

### Validação do nicho (2 ferramentas):
1. **YouTube Search** — pesquise palavras-chave do nicho. Vídeos recentes com 50K-300K views = bom sinal
2. **1of10.com** — filtrar: últimos 6 meses, outlier 2x mínimo, 50K views mínimo, 10min mínimo

## 📅 PASSO 3: Calendário de Conteúdo

### Frequência ideal:
- **2 vídeos longos/semana** (menos = crescimento lento, mais = algoritmo confuso)
- **Shorts** — alimentados pelos Longs cortados (OpusClip, Vidyo.ai)

### Sistema Notion (template do vídeo):
```
Propriedades de cada vídeo:
- Status: Ideia → Script → Filmagem → Edição → Publicado
- Status Thumbnail: Não iniciado / Em andamento / Concluído

Seções de cada vídeo (toggles):
├── 📚 Pesquisa (screenshots outliers)
├── 💡 Conceito (resumo 4 frases)
├── 📌 Títulos (3 variações)
├── 🖼️ Miniaturas (3 conceitos)
└── 📝 Roteiro (bullet points)
```

## 🔥 PASSO 4: Ideias para Vídeos Virais

### Método "Algoritmo Descartável":
1. Criar NOVA conta YouTube (descartável)
2. Pesquisar 6-7 concorrentes do nicho
3. Abrir 6-7 vídeos populares deles em abas novas
4. Deixar TODOS rodando simultaneamente (mínimo 30 segundos cada)
5. Fechar abas e ir para o home do YouTube
6. Instalar extensão 1of10 Chrome (gratuita)
7. Navegar e printar vídeos outliers da home

### Validar outlier (1of10.com):
- Filtros: últimos 6 meses, outlier 2x, 50K views mínimo, 10min mínimo
- Pesquisar pelo tema do outlier que encontrou
- Se encontrar 4+ outliers sobre o mesmo tema nos últimos 6 meses = FAZER ESSE VÍDEO

### Diferencial obrigatório:
Nunca copie exatamente — adicione seu twist único:
- Adicionar IA ao tema (está em alta em 2026)
- Fazer versão mais longa (ex: 1 hora vs 15 min)
- Seu ângulo de marca pessoal
- Informação mais atualizada

## ✍️ PASSO 5: Roteiro (Bullet-Point, não palavra por palavra)

### Por que bullet-points (não roteiro completo)?
- Mais natural e autêntico
- Evita parecer robótico lendo roteiro
- Mais rápido de produzir
- Permite improviso que soa mais humano

### Prompt exato — roteiro:
```
Escreva um roteiro em tópicos para este vídeo do YouTube.
Em tópicos (bullet points), NÃO em frases completas.
Hook com duração máxima de 30 segundos.

[RESUMO DO CONCEITO]

Títulos considerados:
1. [TÍTULO 1]
2. [TÍTULO 2]
3. [TÍTULO 3]
```
Depois: "Adicione mais tópicos a isso. Seja mais detalhado para estar totalmente pronto para gravar."

### Prompt — conceito (4 frases):
```
Escreva um resumo conceitual de 4 frases para um vídeo do YouTube sobre:
[COLE TRANSCRIÇÃO DO ÁUDIO COM SUAS IDEIAS]
```

### Prompt — títulos (10 variações):
```
Escreva 10 variações de título para este vídeo.
Dê uma nota de 1 a 10 para clickabilidade.
Sem travessões. Recomende o top 3.
[RESUMO DO CONCEITO]
```

### Prompt — hooks (3 variações):
```
Escreva 3 variações de hook para este vídeo:
- Afirmação ousada
- Dor com a qual o público se identifica
- Fato surpreendente

Formato de lista com marcadores, não frases completas.
[CONCEITO + TÍTULO]
```

## 📤 PASSO 7: Publicação e Otimização (Configurações Exatas 2026)

### Checklist de upload obrigatório:
- ❌ NUNCA marcar "Conteúdo para crianças"
- ✅ Sempre definir **categoria** do vídeo

- ✅ Adicionar **capítulos** (usar Descript AI ou manualmente)
- ✅ Primeiro link da descrição = seu produto/oferta/lista de emails
- ✅ Fixar **comentário** logo após publicar (pergunta + link útil)
- ❌ Tags — não fazem diferença, ignore
- ❌ Testes A/B — só com média de 3.000+ views/vídeo

### Prompt — comentário fixado:
```
Escreva um comentário fixado para este vídeo do YouTube
com uma pergunta para aumentar o engajamento:
[TÍTULO + CONCEITO DO VÍDEO]

- O que importa NÃO é CTR global — é CTR nas primeiras 24h vs outros vídeos seus

## 📊 PASSO 8: Analytics (O que realmente importa)

### O que NÃO importa:
- CTR global (cai naturalmente com mais alcance)
- Número de inscritos
- Likes e comentários isolados

### O que IMPORTA:
| Métrica | Alvo |
|---------|------|
| Retenção primeiros 30s | 60-70% |
| CTR nas primeiras 24h | comparar com seus outros vídeos |
| Gráfico de retenção | deve ser plano (sem quedas bruscas) |
| Divisão novos vs recorrentes | maioria deve ser NOVOS espectadores |

### Prompt — análise:
```
Analise os padrões nos meus últimos [X] vídeos e identifique:
- O que está funcionando

## 💰 PASSO 9: Monetização (Três Níveis)

### Nível 1 — AdSense:
- 1.000 inscritos + 4.000h assistidas nos últimos 365 dias
- RPM médio: $2-10/1K views (varia por nicho e geo)
- **NÃO é renda principal** — é bônus

### Nível 2 — Patrocínios:
- $500-$10.000 por vídeo dependendo do canal
- Usar Claude para: e-mail de prospecção, media kit, preços, responder marcas

### Nível 3 — Produto/Serviço Próprio (MAIOR ROI):
- Curso, e-book, mentoria, comunidade, software, SaaS
- Canal gera tráfego gratuito qualificado para o produto
- Primeiro link da descrição = oferta principal
